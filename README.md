# 📊 COVID-19 Global Data Visualization Analysis

## 📌 Project Summary

The **COVID-19 Global Data Visualization Project** is a data-driven analytical study developed using Python to explore global pandemic trends through visual analytics.

This project focuses on transforming COVID-19 datasets into meaningful visual insights such as global case growth, country-wise impact, death and recovery patterns, and relationships between key pandemic indicators.

Through multiple visualization techniques, the project highlights how COVID-19 spread globally and how different countries were affected over time.

---

## 🎯 Project Objectives

* Visualize global COVID-19 case trends over time
* Identify the most affected countries
* Analyze relationships between confirmed cases and deaths
* Explore distribution of active cases across countries
* Compare COVID-19 trends between major countries
* Present insights through clear and informative visualizations

---

## 🛠 Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

---

## 📊 Key Visualizations Performed

### 🔹 Global COVID-19 Trend

* Line chart visualization of confirmed, deaths, and recovered cases over time
* Helps understand pandemic growth globally

### 🔹 Country-wise Case Analysis

* Top 10 countries with the highest confirmed cases visualized using bar charts
* Identifies the most impacted regions

### 🔹 Global Case Distribution

* Pie chart representing proportion of confirmed, deaths, and recovered cases
* Shows overall pandemic composition

### 🔹 Confirmed vs Deaths Relationship

* Scatter plot visualization between confirmed cases and deaths
* Demonstrates correlation between infection and mortality

### 🔹 Active Cases Distribution

* Histogram showing distribution of active cases among countries
* Highlights variation in case intensity across regions

### 🔹 Correlation Heatmap

* Heatmap illustrating relationships between confirmed, deaths, recovered, and active cases
* Helps identify strong statistical correlations

### 🔹 Country Comparison Analysis

* Multi-line chart comparing COVID-19 trends between India and the United States
* Demonstrates differences in case growth patterns

---

## 📈 Key Insights

* COVID-19 cases increased significantly worldwide during the pandemic period
* Some countries experienced much higher case counts than others
* Confirmed cases and deaths show strong correlation
* Recovery trends improved gradually across many regions
* Data visualization simplifies complex global health data for easier interpretation
